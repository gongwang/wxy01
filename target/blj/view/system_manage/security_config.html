*锁定时长：  分钟  

有效值0-10080。如果设置为0，则锁定用

密码策略配置

密码强度校验：

开启强密码校验，密码长度为8-32个字符，
且包含大小写字母、数字和特殊字符，不支持空格

新用户强制改密：

本地认证用户首次登录系统后必须修改密码

*密码相同校验：  次  

有效值1-30。检查新密码不能与前N次
 设置的密码相同

*密码修改周期：  天  

有效值0-365。如果设置为0，则账户密码
永不过期，否则当达到修改周期后须修改密码
